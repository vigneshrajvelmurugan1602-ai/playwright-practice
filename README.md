# Playwright Automation Testing Framework

A professional end-to-end test automation framework built with Playwright and JavaScript.

## 🛠️ Tech Stack
- Playwright
- JavaScript
- Node.js
- Page Object Model (POM)

## 📁 Project Structure
## ✅ Test Cases Covered
- Page title verification
- Navigation and link testing
- Positive login testing
- Negative login testing (invalid credentials)
- Codegen recorded tests

## 🚀 How to Run Tests

Install dependencies:
```bash
npm install
npx playwright install
```

Run all tests:
```bash
npx playwright test --project=chromium
```

Run with browser visible:
```bash
npx playwright test --project=chromium --headed
```

Run in UI mode:
```bash
npx playwright test --ui
```

## 📊 Test Report
```bash
npx playwright show-report
```
